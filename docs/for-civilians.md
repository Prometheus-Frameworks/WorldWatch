# For Civilians

WorldWatch is a public-source awareness tool. It helps you quickly understand:

- **What changed** in current regional signals.
- **Why it matters** for civilian life (safety, energy, logistics, displacement pressure).
- **What to watch next** before drawing stronger conclusions.

## What WorldWatch is not

- Not a prediction engine.
- Not a military targeting tool.
- Not a substitute for local emergency guidance or legal advice.

## How to read the civilian cards

1. Start with the **What changed** line for movement direction and magnitude.
2. Read **Why it matters** for practical impact framing.
3. Use **What to watch** for confirmation checks, especially when freshness or confidence is low.

## Data honesty

If data is missing, stale, or incomplete, WorldWatch should say so directly.
